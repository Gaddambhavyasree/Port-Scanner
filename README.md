# Mini Port Scanner 🔍

A simple TCP port scanner written in Python.  
This tool checks whether ports are open or closed on a target host and displays the associated service name.

## Features

- Scan a range of TCP ports
- Detect open and closed ports
- Display common service names (FTP, SSH, HTTP, etc.)
- Timeout handling to avoid hanging connections
- Lightweight and easy to use

## Requirements

- Python 3.x

No external libraries are required.

## How to Run
1. Clone the repo
2. Give the excecutable permissions
3. Run the file with ./filename
4. After running take any sample ip (Metasploit) and enter the ip address and Starting and ending ports to scan.
